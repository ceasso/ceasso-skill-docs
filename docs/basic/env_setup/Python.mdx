# Python环境配置


## Python介绍
想象一下，当其他编程语言还在为“少写一个分号”跟你吵架时，Python 正翘着二郎腿说：“兄弟，放松点，括号和分号？咱这儿能省则省～”

作为编程语言界的“社交达人”，Python 有三大必杀技：
- **语法像说话一样自然**：同样是“打印 hello world”，有的语言要写三行声明，Python 只需要 `print("hello")` —— 简直是“编程界的白话文”
- **兼容性堪比万能插座**：不管你用 Windows、Mac 还是 Linux，它都能轻松适配，甚至在树莓派、服务器上也能蹦跶
- **“轮子”多到能开工厂**：数据分析、游戏开发、人工智能... 别人造好的工具直接拿过来用，效率拉满！

简单来说，学 Python 就像骑共享单车——不用自己造轮子，还能轻松到达目的地～


## 🛠️ Python 环境配置指南

### 第一步：安装 Python 解释器
就像玩游戏需要游戏机，运行 Python 代码也需要“解释器”这个“机器”。

1. 打开官网：[python.org/downloads](https://www.python.org/downloads)（认准带官方标识的链接哦）
2. 点击“Download Python X.X.X”（X 是版本号，选最新的稳定版就行）
3. 安装时务必勾选 **“Add Python to PATH”**（重点！不然电脑找不到这个“机器”）
4. 一路点“Next”，最后点“Install”，等待进度条走完


### 第二步：安装 Visual Studio Code（简称 VS Code）
这是咱们写代码的“笔记本”，轻便又好用。

1. 打开官网：[code.visualstudio.com](https://code.visualstudio.com)
2. 点击“Download”，根据你的电脑系统（Windows/Mac）选择对应版本
3. 安装过程很简单，默认选项一路下一步就好

:::tip 💡
### 这里的开发工具大家也可以使用更加专业的PyCharm
### Windows系统
- 从[PyCharm官方下载页面](https://www.jetbrains.com/pycharm/download/)下载`.exe`安装程序，对于ARM64处理器，需选择相应的安装程序。可使用下载页面中链接的SHA校验和验证安装程序的完整性。
- 运行安装程序，按照向导步骤进行安装。注意安装向导中的选项，如“64位启动器”可添加桌面启动图标；“将文件夹打开为项目”会在文件夹上下文菜单中添加相关选项；“.py”可建立与Python文件的关联；“将launchers目录添加到PATH”可允许从控制台运行PyCharm实例。
- 安装完成后，可在Windows开始菜单中找到PyCharm，或使用桌面快捷方式运行，也可运行安装目录中`bin`下的启动批处理脚本或可执行文件。
### macOS系统
- 从[PyCharm官方下载页面](https://www.jetbrains.com/pycharm/download/)下载磁盘映像，Intel和Apple Silicon处理器有单独的磁盘映像，可通过SHA校验和验证完整性。
- 将镜像挂载后，把PyCharm应用程序拖动到“应用程序”文件夹。
- 可从“应用程序”目录、Launchpad或Spotlight运行PyCharm应用程序。
### Linux系统
- 从[PyCharm官方下载页面](https://www.jetbrains.com/pycharm/download/)下载适用于ARM64处理器或其他版本的tarball`.tar.gz`文件。
- 将`pycharm - *.tar.gz`文件解压到其他文件夹，命令为`tar xzf pycharm - *.tar.gz`。
- 进入解压后的`bin`目录，运行`pycharm.sh`即可启动PyCharm。

### 第三步：给 VS Code 装个“Python 插件”
就像给笔记本装个“Python 专用笔”，让它能识别 Python 代码。

1. 打开 VS Code，左边栏点那个“方块”图标（扩展商店）
2. 在搜索框输入“Python”，找到第一个带“Microsoft”图标的插件
3. 点击“Install”安装，安装完成后重启 VS Code 生效


### 第四步：测试配置是否成功
来个“开机仪式”，看看咱们的工具能不能正常工作～

1. 打开 VS Code，新建一个文件（快捷键 Ctrl+N）
2. 输入一行代码：`print("Python 配置成功！我超勇的！")`
3. 按 Ctrl+S 保存，文件名写 `test.py`（.py 是 Python 文件的“身份证”）
4. 右键点击编辑器，选择“Run Python File in Terminal”
5. 如果看到终端里显示“Python 配置成功！我超勇的！”，恭喜你——搞定啦！


### 小提示：遇到问题别慌
如果步骤中出错了，先检查这两点：
- 安装 Python 时有没有勾选“Add to PATH”？没勾的话可以重新安装一次
- VS Code 里的 Python 插件是不是最新版本？可以在扩展商店点“更新”
